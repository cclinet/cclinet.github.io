基于 Pocket Flow 强大的图（Graph）抽象，实现各种流行的设计模式变得异常简单。无论是构建复杂的**多智能体（Multi-Agent）**系统，还是设计精细的**工作流（Workflow）**，亦或是实现**检索增强生成（RAG）**，Pocket Flow 都能轻松应对。

以下是 Pocket Flow 支持的一些核心设计模式：

- [**多智能体 (Multi-Agent)**](https://the-pocket.github.io/PocketFlow/design_pattern/multi_agent.html): 协调多个智能体协同工作。
- [**智能体 (Agent)**](https://the-pocket.github.io/PocketFlow/design_pattern/agent.html): 构建具备自主能力的智能体。
- [**工作流 (Workflow)**](https://the-pocket.github.io/PocketFlow/design_pattern/workflow.html): 定义清晰的任务执行流程。
- [**RAG**](https://the-pocket.github.io/PocketFlow/design_pattern/rag.html): 结合外部知识增强生成能力。
